# ABS Diagnostics

Step-by-step guide to diagnose Anti-lock Braking System issues.

## Common Issues

| Symptom | Possible Cause |
|---------|----------------|
| ABS Light On | Sensor failure |
| Pulse when braking | Wheel speed sensor |
| No ABS function | Control module |

## Tools Required

- OBD-II Scanner
- Multimeter
- Scan Tool

## Step-by-Step

1. Connect OBD-II scanner
2. Check for fault codes
3. Test wheel speed sensors
4. Inspect control module wiring
5. Clear codes and test drive
